## Project Overview

This repository contains a complete data architecture solution for the FlexiMart system. It includes a relational database ETL pipeline with schema documentation, business queries, and data quality checks, along with a NoSQL component featuring MongoDB operations and a product catalog design. The solution demonstrates structured data processing, analytics readiness, and scalable data modeling.
